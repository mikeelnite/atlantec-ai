# Datasets

Place the downloaded datasets here.

Expected files:
- `gael_towns.geojson` — your custom list of Gaeltacht towns
- `Gaeltacht Boundaries Generalised 100m.geojson` (optional)
- `Settlements Generalised 100m.geojson` (optional)

These files are published by Tailte Éireann under a Creative Commons Attribution license.

If you prefer remote access, you can also query the ArcGIS GeoService REST endpoints directly.
